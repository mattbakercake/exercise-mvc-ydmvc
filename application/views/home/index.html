Home Page

This site is a simple demo of the MVC framework. It contains a single default controller (Home_Controller).

There are 3 models: Home (which belongs to the Home_Controller), Fruit (defines fruit properties and methods) and User (defines user properties and methods).

The default controller and action (index()) can be run by either visiting the root of the site ()or by specifying the controller and action in the URL ().

Use the navigation buttons above to trigger the different controller actions.
