Class FrontController

The FrontController class is the heart of the framework interpreting requests and dispatching appropriate controllers

It determines whether controller/action/parameters have been passed to it and either sets these values or calls the Router class to extract the values from the url or serve a stored route. The run method dispatches the controller.
