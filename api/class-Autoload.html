Class Autoload

The Autoload class is a static class that contains methods to automatically load class files when required by the application

The methods are added to the SPL_autoload_register to be registered as autoload implementations and look for classes in different areas of the framework

Category: Core
Author: Matt Baker <dev@mikesierra.net>
Version: 0.1
Since: 26-02-2013
Example: spl_autoload_register('Autoload::coreLoader');
Located at application/ydmvc_core/Autoload.php

# coreLoader( string $className )

Autoloads classes in the framework's core directory. looks for a file with the same name as the class being called; the class and filename should match and be title case e.g. Example.php
